# sql-data-warehouse-project
This project demonstrates a comprehensive way to model data warehousing and analystics solutions, mainly revolving around building a data warehouse while highlighting mordern industry standards. 

**Building the data warehouse:**
- Objective: To develop a modern data warehouse that consolidates sales data using SQL server for enhanced reporting and decision making
- data sources: importing data from two source systems (ERP and SRM) provided as CSV files.
- data quality: cleanse and resolve data quality issues before analysis.
- integration: combine both sources into a single data model
- scope: to focus on latest dataset only.
